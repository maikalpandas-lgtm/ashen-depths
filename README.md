# Ashen Depths

Fantasy 1st-person dungeon crawl + turn-based **card combat**.  
Random labyrinths, party of 3, Diablo/Warcraft vibe. Cooking/craft is secondary.

**Engine:** Godot 4.7+  
**Design doc:** [docs/DESIGN.md](docs/DESIGN.md) · **Roadmap:** [docs/ROADMAP.md](docs/ROADMAP.md) · **Handoff:** [docs/HANDOFF.md](docs/HANDOFF.md)

## Run

1. Install [Godot 4.7+](https://godotengine.org/)
2. Open this folder as a project
3. Run `scenes/main.tscn` (F5)

### Controls
Grid crawler — camera is locked forward.

- **W / S** — step one tile forward / back  
- **A / D** — turn 90°  
- **R** — regenerate dungeon (debug)  
- **Esc** — release/capture mouse
