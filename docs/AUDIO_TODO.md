# Звук — что ещё нужно достать

Код уже ищет эти файлы. Пока их нет, игра **не молчит**: любой отсутствующий
клип откатывается на пещерный аналог (`Sfx._pick_stream`), а отсутствующий
эмбиент просто не играет. Так что класть можно по одному, в любом порядке.

Формат: **`.ogg`**, моно для точечных звуков, стерео для эмбиента.
Класть в `assets/audio/sfx/`. Имя файла = id из таблицы, ровно.

Лицензия — только CC0 / OFL-подобное (как весь текущий Kenney-набор,
см. `KENNEY_LICENSE.txt`). Источник каждого файла записывать в `README.md`
рядом, иначе через месяц не докажем происхождение.

## Лес (`biome = "forest"`)

| Файл | Что | Длина | Заметка |
|---|---|---|---|
| `forest_step_0..3` | Шаг по листве / траве | 0.2–0.4 с | Нужны **4 варианта**, движок берёт случайный + питч ±6%. Один клип на повторе слышно сразу. |
| `forest_bump` | Упёрся в дерево | 0.2–0.3 с | Глухой деревянный, не каменный |
| `forest_chest` | Открыть схрон | 0.5–1 с | Скрип дерева вместо лязга |
| `forest_path` | Уход на след. уровень | 1–2 с | Шелест / шаги вдаль |
| `amb_forest` | **Эмбиент леса** | 30–60 с, бесшовно | Птицы, ветер в кронах, изредка скрип. Играет на шине SFX, громкость −14 dB |

## Требование к эмбиенту

Зацикливается движком (`loop = true`), поэтому **шов обязан быть незаметен**:
никакого затухания в конце файла и никакого выраженного события (одиночный
крик птицы, удар) у самых краёв — на повторе оно превратится в метроном.

## Как проверить

```bash
godot --headless --script tests/sfx_biome_test.gd
```

Тест ловит главную ловушку: биомный клип, у которого **нет базового аналога**
в `CATALOG`. Такой id не откатывается никуда и даёт тишину — а тишину на
плейтесте не замечают, пока не пропадёт весь пак.
