# Ashen Depths — Design Document

> Живой документ. Возвращайся сюда за планом, лором и решениями по геймплею.  
> Статус: **Фаза 0–1 (скелет + лабиринт MVP)**  
> Движок: **Godot 4.7** · Стиль: cartoon dark fantasy (реф UI/камера — card dungeon crawler)  
> Репо: `ashen-depths`

---

## 1. One-liner

**1st-person dungeon crawl** + **походовый карточный бой** в лабиринтах, которые **каждый раз генерируются заново**.  
Партия из **3 героев**, лут и боссы в духе **Diablo / Warcraft**.  
Готовка/крафт — **второстепенно** (хаб, зелья), не win-condition.

---

## 3. Лор (зачем лезем в лабиринты)

### Мир
**Аэтерн** держался на **Трёх Печатях**, запиравших **Бездну Пепла** (Ashen Abyss).  
Печати вросли в **Лабиринты Корня** — живой камень, который **перестраивается каждый цикл**  
(= рандом-генерация — часть мира, не только механика).

### Антагонист
**Король-маг Валор** попытался украсть силу Печатей → стал **Повелителем Пепла**.  
Одна Печать треснула → твари, пепел, падение окраин.

### Игрок
Наёмник-предводитель. Нанимает **3 героев**. Заказчик — **Серый Совет** (остатки Ордена + гильдии).

### Цель
1. Спускаться по этажам Лабиринтов Корня  
2. Собирать **осколки Печати** (боссы / элиты)  
3. Дойти до **Трона Пепла** → закрыть Печать **или** убить Валора (ветки концовок)

### Концовки (позже)
- **Страж** — восстановить Печать  
- **Узурпатор** — корона Валора  
- **Изгой** — разрушить все Печати (NG+)

## 4. Геймплей-столпы

### 4.1 Исследование (Doom-коридоры)
- 1st person, WASD + мышь  
- Рандомный лабиринт: **комнаты + коридоры**, тупики, развилки  
- **Потайные стены / лазейки**, сундуки, элитные комнаты  
- Факелы, туман, миникарта (как в рефе)  
- Босс-комната достижима; shortcut — риск/награда  

### 4.2 Бой (как на скрине рефа)
- **Не** real-time шутер  
- **Пошаговый**, рука **карт**, ресурс **энергии** (⚡ 0–3 на старте)  
- Играешь карты → **END TURN** → ходят враги  
- После **пачки** → выбор **1 из 3 карт** в колоду рана  
- После **level up** → более сильный выбор  

### 4.3 Партия
- **3 героя** до спуска (MVP: **одна общая колода** отряда)  
- Портреты / HP; позже — переключение «кто бьёт»  
- Глубже (пост-MVP): колода на героя  

### 4.4 Крафт (вторично)
- Верстак в хабе: зелья, заточка, свитки  
- Не обязателен для прохождения

## 6. Цикл одного спуска (run)

```
ХАБ (лагерь Совета)
  → выбор 3 героев / сложности
  → GENERATED LABYRINTH (seed)
  → исследование коридоров
  → пачка мобов → КАРТОЧНЫЙ БОЙ
  → победа → 3 карты, взять 1
  → сундуки / тупики / секреты / элиты
  → БОСС ЭТАЖА → осколок / ключ вниз
  → следующий этаж или возврат в хаб
```

## 7. Система карт (черновик)

**Типы:** Attack · Guard · Skill · Spell · Relic (пассивка) · Event  

**Примеры MVP:**

| Карта | ⚡ | Эффект |
|-------|---|--------|
| Slash | 1 | 6 урона |
| Hack | 2 | 10 урона |
| Shield | 1 | +5 Block |
| Firebolt | 1 | 5 урона, ignore 1 armor |
| Rally | 0 | +1 ⚡ next turn (limit) |

После пачки / уровня — draft как Slay the Spire.

## 8. Генерация лабиринта (технически)

### Алгоритм MVP (Фаза 1)
1. Сетка `W × H` (например 41×41 клеток)  
2. Расставить `N` прямоугольных **комнат** без сильного overlap  
3. Соединить **MST / sequential L-corridors**  
4. Пометить **тупики** (клетки коридора с 1 соседом)  
5. **Двери** — на стыке room↔corridor  
6. **Сундук** — в 1 тупике или дальней комнате  
7. **Факелы** — вдоль стен с шагом  
8. **Пачки** — в N комнатах (не старт), маркер Encounter  
9. **Босс-комната** — самая дальняя от старта (Фаза 2 можно усилить)

### Содержимое клетки
`EMPTY | FLOOR | WALL | DOOR | CHEST | TORCH | ENCOUNTER | START | EXIT`

## 12. Решения, которые зафиксированы

1. Бой = **карты + энергия + end turn** (не Doom-shooter).  
2. Исследование = **1st person коридоры**.  
3. Крафт/еда = **не core**.  
4. Партия MVP = **3 героя, 1 колода**.  
5. Арт = cartoon dark fantasy; генерация AI ок, нужен style lock.  
6. Название рабочее: **Ashen Depths** (можно сменить).
