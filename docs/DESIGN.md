# Ashen Depths — Design Document

> Живой документ. Возвращайся сюда за планом, лором и решениями по геймплею.  
> Статус: **Фаза 0–1 done · combat (§7 Inscryption-cards, no board) + backpack BB (§8) зафиксированы**  
> Движок: **Godot 4.7** · Стиль: cartoon dark fantasy (реф UI/камера — card dungeon crawler)  
> Репо: https://github.com/maikalpandas-lgtm/ashen-depths

---

## 1. One-liner

**1st-person dungeon crawl** + **походовый карточный бой** в лабиринтах, которые **каждый раз генерируются заново**.  
Партия из **3 героев**, лут и боссы в духе **Diablo / Warcraft**.  
Готовка/крафт — **второстепенно** (хаб, зелья), не win-condition.

## 2. Отличие от рефа «Подземелья вкусностей»

| Реф | Ashen Depths |
|-----|----------------|
| Ингредиенты + блюда = ядро | **Осколки Печати + Diablo-лут** |
| Food-horror aesthetic | **Dark fantasy** (можно тот же cartoon outline) |
| Крафт еды в центре | Крафт только support |
| Каркас геймплея | **Берём:** 1st person + карты + END TURN + миникарта |

## 3. Лор (зачем лезем в лабиринты)

> 🔄 **22.07.2026 — сеттинг переведён в славянскую мифологию.** Старый набор
> (Каэль / Лира / Сера, пещерные твари) **не выброшен**: это верхние этажи, копи.
> Славянское начинается с 3-го этажа, в Нави. Причина: визуально мы шли слишком
> близко к «Подземельям вкусностей», а прежний лор был взаимозаменяем с любым
> фэнтези-рогаликом.

### Мир
**Явь** держится на **Трёх Зароках**, запирающих **Навь** — мир мёртвых.
Зароки вросли в **Лабиринты Корня** — корни Мирового Древа между Явью и Навью,
живой камень, который **перестраивается каждый цикл**
(= рандом-генерация — часть мира, не только механика).

### Антагонист
**Кощей Бессмертный** тянет силу Зароков → один треснул, Навь пошла вверх.
Его смерть спрятана слоями: **игла → яйцо → утка → заяц → сундук** — это и есть
структура забега: каждый этаж снимает одну оболочку. Осколки Зарока с боссов =
шаги к игле.

### Игрок
Наёмник-предводитель. Нанимает **3 героев**. Заказчик — **Ведовской круг** (волхвы и старшины).

### Цель
1. Спускаться по этажам Лабиринтов Корня  
2. Собирать **осколки Печати** (боссы / элиты)  
3. Дойти до **Кощеевой иглы** → запереть Зарок **или** сломать иглу (ветки концовок)

### Концовки (позже)
- **Страж** — восстановить Печать  
- **Узурпатор** — корона Валора  
- **Изгой** — разрушить все Печати (NG+)

## 4. Геймплей-столпы

### 4.1 Исследование (Doom-коридоры)
- 1st person, WASD + мышь  
- Рандомный лабиринт: **комнаты + коридоры**, тупики, развилки  
- **Потайные стены / лазейки**, сундуки, элитные комнаты  
- Факелы, туман, миникарта (как в рефе)  
- Босс-комната достижима; shortcut — риск/награда  

### 4.2 Бой (зафиксировано: FPS + рука карт, **без board**)
- **Не** real-time шутер, **не** Inscryption-стол / сетка существ  
- **Пошаговый** поверх 1st-person коридора (как на скрине рефа)  
- **Враги** стоят **в сцене** (3D/billboard) с HP, intent, block  
- **Рука карт** внизу, **END TURN**, ресурс **⚡** (MVP: общие **3** на ход)  
- **Карты** — механики/«сигилы» в духе **Inscryption** (см. §7), но играются **из руки** в цель  
- Прогрессия колоды — **3 слоя** (§7.6)  

### 4.3 Партия
- **3 героя** до спуска  
- **У каждого своя колода** и пул карт под роль  
- UI боя: **одна общая рука**; карты **цвет / портрет героя**; ⚡ общая **3** (позже: 1+1+1 если нужно)  
- На экране: HP партии (или фокус-герой + party bar), ⚡, draw/discard counts  

### 4.4 Рюкзак / магазин (Backpack Battles)
- **Карты** = ходы в бою  
- **Рюкзак** = пассивы / adjacency / синергии (не auto-battler)  
- Дроп + **shop после этажа** + **0–1 merchant** на карте  
- Полный дизайн → **§8**

### 4.5 Крафт / лагерь (вторично)
- Верстак: зелья, заточка  
- **Camp services (Фаза 4):** remove / upgrade карты за золото (как StS)  
- Не обязателен для прохождения

## 6. Цикл одного спуска (run)

```
ХАБ (лагерь Совета)
  → выбор 3 героев / сложности · расстановка рюкзака
  → GENERATED LABYRINTH (seed)
  → исследование коридоров (± 1 MERCHANT на этаже)
  → пачка мобов → КАРТОЧНЫЙ БОЙ
  → победа → DRAFT СЛОЙ 1 (карта ИЛИ skip→gold)
         → затем LOOT ITEM (take / sell)   ← слой рюкзака
  → (если level-up) → DRAFT СЛОЙ 2 (upgrade / rare)
  → сундуки / тупики / секреты / элиты
  → БОСС ЭТАЖА → DRAFT СЛОЙ 3 (relic + shard + optional curse altar)
              → босс-предмет в рюкзак (rare/epic pick)
  → FLOOR SHOP (витрина, reroll, sell, optional card remove)
  → следующий этаж или возврат в хаб
  → (хаб) remove/upgrade карт + edit backpack
```

## 7. Combat + карты (зафиксировано)

### 7.0 Гибрид (что берём / что **не** берём)

| Источник | Берём | **Не** берём |
|----------|--------|----------------|
| **Реф-скрин (card dungeon)** | Враги **в коридоре**, HP/intent, рука внизу, ⚡, **END TURN**, block | — |
| **Inscryption** | Карты-«существа/ритуалы» с **cost**, **sigils** (keywords), **sacrifice/blood** как cost, вкус dark-table | **Board / lanes / слоты существ на столе**, карта-как-пешка на сетке |
| **Slay the Spire** | Draw / hand / discard, energy, end turn, intents | Полный 2D map-only loop |
| **Backpack Battles** | Только **рюкзак** (§8), не бой | Auto-battler вместо карт |

**Формула боя:**  
`FPS-коридор + billboard-враги` · `рука карт (Inscryption-flavor)` · `END TURN` · `рюкзак пассивит карты (BB)`.

### 7.1 Presentation боя (как на скрине)
При `ENCOUNTER` → combat overlay, камера **lock**, движение **off**:

```
[ миникарта / party ]     ENEMY 1    ENEMY 2    ENEMY 3     [ intent / preview ]
                          HP bar     HP bar     HP bar
                          🗡 intent  🛡 block   …
[ ⚡ 0/3 ]  [ hand: 3–5 cards ]                              [ END TURN ]
           draw pile · discard
```

| Элемент | Правило MVP |
|---------|-------------|
| Враги | **1–4** в ряд в глубине коридора (billboard / mesh), клик = target |
| Target | По умолчанию **левый живой** / last selected; AoE = все / front row |
| Party HP | **Общий пул HP отряда** (проще) **или** 3 HP-бара (если успеем) — **MVP: общий HP** |
| Block | Общий **Block** партии до конца хода врагов (сгорает в начале своего хода, как StS) |
| Intent | Иконка + число над врагом (атака / блок / баф / дебаф) |
| Turn | Player acts (карты) → **END TURN** → resolve enemy intents → draw |

### 7.2 Типы карт (Inscryption-flavor, **из руки**)

| Тип | Что делает | Cost |
|-----|------------|------|
| **Strike** | Урон 1 цели | ⚡ |
| **Guard** | Block | ⚡ |
| **Skill** | Баф / draw / debuff | ⚡ |
| **Spell** | Магия, часто AoE / ignore armor | ⚡ |
| **Blood rite** | Сильная карта: cost = **потеря HP** партии (или ⚡ + HP) | 🩸 / ⚡+🩸 |
| **Sigil card** | Карта с **1–2 sigils** (keywords) | ⚡ |
| **Curse** | Негатив в колоде (алтарь) | — |
| **Relic** | Не в колоде — пассивка забега | — |
| **Shard** | Мета между забегами | — |

**Явно out of scope:** выкладывать карты на 3–4 клетки «стола»; существа, которые остаются на board между ходами как в Inscryption.

### 7.3 Sigils (keywords) — Inscryption → эффекты карт

Сигил = **тег на карте** (иконка + правило). MVP-набор:

| Sigil | Эффект при розыгрыше / условии |
|-------|--------------------------------|
| **Sharp** | +2 урона, если цель с Block |
| **Pierce** | Игнор 50% Block (или flat ignore 3) |
| **Sweep** | Урон делится на всех врагов **или** full AoE с −30% |
| **Drain** | Урон → heal party на 50% нанесённого (cap) |
| **Frail hit** | Если цель ≤40% HP → +50% урон |
| **Bone** | При убийстве этим ударом: **+1 Bone** (ресурс, см. ниже) |
| **Sanguine** | Cost включает **HP** (blood) |
| **Echo** | Срабатывает ещё раз за 0⚡, если есть **Bone ≥1** (съедает 1 Bone) |
| **Ward** | Даёт Block **и** 1 Thorns до след. хода |
| **Cleave** | Урон главной цели + 50% соседней (слева/справа в ряду) |

**Bone (кости)** — опциональный combat-ресурс MVP+: старт 0; с kills / карт; тратится Echo/особыми картами.  
**Не** путать с Bone token board Inscryption — только **счётчик на UI боя**.

### 7.4 Стоимость и розыгрыш

| Правило | MVP |
|---------|-----|
| Основной resource | **⚡** — **3** в начале хода игрока |
| Blood cost | Карта с 🩸: платишь **N HP** партии **при розыгрыше** (можно + ⚡) |
| Sacrifice (упрощ.) | **Не** жертвуем существ на board. Вместо: карта **«Offering»** — discard 1 другую карту из руки → скидка / усиление |
| Targeting | Strike/Skill → 1 enemy; Sweep/Cleave по правилам; Guard/Skill self — без target |
| Unplayable | Не хватает ⚡/HP → карта серая |

### 7.5 Колоды

| Правило | Значение |
|---------|----------|
| Колод на забег | **3** (по герою) |
| Старт | **8–10** карт / герой |
| Draw | **5** в начале боя; **+5** (или до hand size 10) в начале хода — **MVP: draw 5, hand cap 10** |
| Soft comfort | ~15–18 карт в колоде героя |
| Hard pressure | >20–22 → remove в shop/hub |
| Remove | Хаб / floor shop за gold (Фаза 4) |

В UI **одна рука** = merge draw из **активного** героя **или** round-robin из трёх колод.  
**MVP-решение:** все 3 колоды шаффлятся в **один combat deck** с **owner-tag** на карте (цвет).  
Позже: ходы «фокус-героя» если нужно.

### 7.6 Три слоя draft (экраны не смешивать)

#### Слой 1 — после пачки
- **2–3** карты → взять **1** в колоду владельца  
- **Или skip → gold** (minor relic roll ~10% опционально)  
- **Отдельно** (след. экран): item loot take/sell (§8)

#### Слой 2 — level-up
- **Upgrade** карты **или** rare/uncommon class card  

#### Слой 3 — босс этажа
- Relic pick **1 of 2–3**  
- Shard **auto**  
- Optional curse altar (power card + Curse / downside)

### 7.7 Примеры карт MVP

| Карта | ⚡ | 🩸 | Sigils | Эффект |
|-------|---|-----|--------|--------|
| Slice | 1 | — | — | 6 урона |
| Hack | 2 | — | Sharp | 10 урона |
| Block | 1 | — | — | +5 Block |
| Cleave Cut | 1 | — | Cleave | 5 + 50% соседу |
| Blood Lash | 0 | 4 | Sanguine, Drain | 12 урона, heal ½ |
| Bone Rattle | 1 | — | Bone | 4 урона; kill → +1 Bone |
| Echo Strike | 1 | — | Echo | 7 урона; (Echo if Bone) |
| Ward | 1 | — | Ward | +4 Block + 1 Thorns |
| Offering | 0 | — | — | Discard 1: next card −1 ⚡ |
| Firebolt | 1 | — | Pierce | 5 урона |

### 7.8 Связь с рюкзаком (BB → карты)
Предметы **не играют карты сами**. Они **модифицируют** розыгрыш:

- `Attack +1` → все Strike  
- `Sigil: Poison touch` → первый Strike/ход вешает Poison  
- Adjacency: клинок + точило → ещё +1 Attack  
- Руна ⚡: 1/бой +1 ⚡  

Баланс-ориентир: **~40% deck · ~40% backpack · ~20% relics**.

### 7.9 Порядок внедрения combat
1. Encounter → freeze explore → 1–3 dummy enemies в сцене + HP/intent  
2. 1 колода, рука, ⚡3, END TURN, Block  
3. 3–5 sigils + 1 blood-карта  
4. Post-combat draft слой 1  
5. 3 колоды merge + цвета  
6. Layer 2–3, bones, shop remove  
7. Backpack hooks (§8)

## 8. Backpack / Shop / Merchant (зафиксировано — **Backpack Battles**)

Вдохновение: **Backpack Battles** (grid, соседство, shop).  
**Не** автобатлер. **Не** board Inscryption.

### 8.1 Разделение ролей

| Система | Роль |
|---------|------|
| **Карты (§7)** | Активные ходы: урон, block, sigils, blood |
| **Рюкзак** | Пассивы / статы / adjacency → **модификаторы карт и статов** |
| **Реликвии** | Крупные правила забега (могут быть «крупным» item later) |
| **Shop / merchant** | Ротация лута, sell, reroll, gold sink |

Формула: `карты = ходы (Inscryption-flavor)` · `рюкзак = билд (BB)` · `враги = 3D-коридор`.

### 8.2 Сетка рюкзака (Ashen Pack)

| Правило | MVP | Позже |
|---------|-----|--------|
| Владелец | **1 сетка отряда 5×4** (20 клеток) | 3 pack по героям |
| Формы | **1×1, 2×1** only | L, 2×2 |
| Rotate | 2×1 можно **повернуть** (1×2) | — |
| Соседство | **Ортогональные 4** (N/E/S/W) | диагонали, aura r=2 |
| Stack | Предметы **не** стакаются в одной клетке | — |
| Редкость | common / rare / epic | legendary |

#### Правила adjacency (конкретно)
- Предмет даёт **base** эффект всегда (если в сетке).  
- **Bonus** срабатывает, если **ортогонально** касается нужного тега/класса item.  
- Пример: `Rusty Blade` base Attack+1; **+1** если сосед `Whetstone`.  
- Цепочки: A–B–C — B сосед и A, и C (оба бонуса, если описано).  
- **Нет** «боя предметов» — только статика + combat hooks при розыгрыше карт / старте хода / старте боя.

#### Примеры предметов

| Предмет | Size | Base | Adjacency bonus |
|---------|------|------|-----------------|
| Rusty Blade | 1×1 | Strike +1 dmg | +1 if next to Whetstone |
| Whetstone | 1×1 | — | enables Blade bonus |
| Shield Shard | 2×1 | Start combat +4 Block | +2 if next to any Guard-tag item |
| Poison Vial | 1×1 | First Strike/turn → 2 Poison | +1 Poison if next to Blade |
| Mana Rune | 1×1 | 1/combat: +1 ⚡ | — |
| Coin Pouch | 1×1 | +15% gold | — |
| Bone Charm | 1×1 | Kills grant +1 Bone (cap 3) | — |
| Blood Charm | 1×1 | Blood cards cost −1 HP (min 1) | — |

### 8.3 Источники предметов

#### A) Дроп после боя
- Обычная пачка: 0–1 common (часто junk)  
- Элита: 1 common/rare  
- Босс: 1 rare/epic **или** pick 1 of 2  

Порядок наград после пачки (два шага, не смешивать в один экран):

```
1) Card draft (слой 1)  OR  skip → gold
2) Item loot             → take into backpack  OR  sell for gold
```

#### B) Магазин после этажа (обязательный)
После босса / перед следующим floor — **full shop**:

- 5–7 предметов на витрине  
- **Reroll** за gold  
- **Sell** из рюкзака  
- (тот же экран / вкладка) card **remove/upgrade** за gold  

#### C) Торговец на карте (0–1 за этаж)
- Отдельная клетка/комната `MERCHANT` (seed рандома)  
- 3–4 слота, цены чуть выше **или** уникальный stock  
- Риск: путь = лишние бои  
- Миникарта: иконка мешка / монет  

```
На этаже:     0–1 wandering merchant
После этажа:  full shop (всегда)
```

### 8.4 UI (эскиз)
- Клавиша **B** / кнопка — экран рюкзака (пауза исследования)  
- Drag-and-drop предметов по сетке  
- Подсветка соседей при наведении (синергии)  
- В магазине: витрина слева, рюкзак справа  

### 8.5 Порядок внедрения (Фаза 4)

| Шаг | Содержание |
|-----|------------|
| **4a** | Inventory list 8 слотов + 2–3 set-бонуса (без grid) — *опциональный прототип* |
| **4b** | Grid 5×4, items 1×1/2×1, adjacency, combat hooks |
| **4c** | Floor shop (buy/sell/reroll) |
| **4d** | Map merchant + minimap icon |
| **4e** | Больше предметов, формы L/2×2, per-hero packs |

**Явно out of scope:**  
- full auto-battler вместо карт  
- предметы, которые **сами** играют карты  
- **Inscryption board** / lanes / permanent creatures on table

### 8.6 Анти-паттерны (не делать)
- Не давать **сильную реликвию + предмет + карту** с каждой пачки без выбора  
- Не раздувать рюкзак так, чтобы draft карт стал бессмысленным (баланс: ~40% сила от deck, ~40% backpack, ~20% relics — ориентир)  
- Не усложнять MVP формами тетромино

## 9. Генерация лабиринта (технически)

### Алгоритм MVP (Фаза 1)
1. Сетка `W × H` (например 41×41 клеток)  
2. Расставить `N` прямоугольных **комнат** без сильного overlap  
3. Соединить **MST / sequential L-corridors**  
4. Пометить **тупики** (клетки коридора с 1 соседом)  
5. **Двери** — на стыке room↔corridor  
6. **Сундук** — в 1 тупике или дальней комнате  
7. **Факелы** — вдоль стен с шагом  
8. **Пачки** — в N комнатах (не старт), маркер Encounter

## 13. Решения, которые зафиксированы

1. Бой = **походовые карты** в **1st-person коридоре** (реф-скрин): враги в сцене, рука, ⚡, END TURN.  
2. **Нет Inscryption board** / lanes / permanent board creatures.  
3. Карты = **Inscryption-flavor**: sigils, blood cost, bones-as-counter, offering-discard — **но из руки**.  
4. Исследование = **1st person** (grid / camera lock — цель feel рефа).  
5. Крафт/еда = **не core**.  
6. Партия = **3 героя**, колоды с owner-tag; MVP combat deck = **merge 3**.  
7. Party HP MVP = **общий пул** + общий Block.  
8. **3 слоя draft:** пачка → level-up → босс (экраны раздельно).  
9. После пачки: card draft **или** skip gold; **отдельно** item take/sell.  
10. **Backpack Battles:** grid **5×4**, items **1×1 / 2×1**, orthogonal adjacency → combat hooks.  
11. **Shop** после этажа + **0–1 merchant** на карте.  
12. Баланс-ориентир: ~40% deck / ~40% backpack / ~20% relics.  
13. Название: **Ashen Depths**.
