# Ashen Depths — Design Document

> Живой документ. Возвращайся сюда за планом, лором и решениями по геймплею.  
> Статус: **Фаза 0–1 done · дизайн прогрессии картами зафиксирован**  
> Движок: **Godot 4.7** · Стиль: cartoon dark fantasy (реф UI/камера — card dungeon crawler)  
> Репо: https://github.com/maikalpandas-lgtm/ashen-depths

---

## 1. One-liner

**1st-person dungeon crawl** + **походовый карточный бой** в лабиринтах, которые **каждый раз генерируются заново**.  
Партия из **3 героев**, лут и боссы в духе **Diablo / Warcraft**.  
Готовка/крафт — **второстепенно** (хаб, зелья), не win-condition.

## 3. Лор (зачем лезем в лабиринты)

### Мир
**Аэтерн** держался на **Трёх Печатях**, запиравших **Бездну Пепла** (Ashen Abyss).  
Печати вросли в **Лабиринты Корня** — живой камень, который **перестраивается каждый цикл**  
(= рандом-генерация — часть мира, не только механика).

### Антагонист
**Король-маг Валор** попытался украсть силу Печатей → стал **Повелителем Пепла**.  
Одна Печать треснула → твари, пепел, падение окраин.

### Игрок
Наёмник-предводитель. Нанимает **3 героев**. Заказчик — **Серый Совет** (остатки Ордена + гильдии).

### Цель
1. Спускаться по этажам Лабиринтов Корня  
2. Собирать **осколки Печати** (боссы / элиты)  
3. Дойти до **Трона Пепла** → закрыть Печать **или** убить Валора (ветки концовок)

### Концовки (позже)
- **Страж** — восстановить Печать  
- **Узурпатор** — корона Валора  
- **Изгой** — разрушить все Печати (NG+)

## 4. Геймплей-столпы

### 4.1 Исследование (Doom-коридоры)
- 1st person, WASD + мышь  
- Рандомный лабиринт: **комнаты + коридоры**, тупики, развилки  
- **Потайные стены / лазейки**, сундуки, элитные комнаты  
- Факелы, туман, миникарта (как в рефе)  
- Босс-комната достижима; shortcut — риск/награда  

### 4.2 Бой (как на скрине рефа)
- **Не** real-time шутер  
- **Пошаговый**, рука **карт**, ресурс **энергии** (⚡ обычно 3 на ход)  
- Играешь карты → **END TURN** → ходят враги  
- Прогрессия колоды — **3 отдельных слоя** (см. §7)  

### 4.3 Партия
- **3 героя** до спуска  
- **У каждого героя своя колода** и свой пул карт под роль (танк / DPS / маг…)  
- Общий фреймворк draft/upgrade/relic одинаковый  
- UI боя (MVP-компромисс): **одна рука внизу**, карты **цвет-кодированы по герою**; энергия **общая 3** или **1+1+1** (решить на прототипе)  

### 4.4 Крафт / лагерь (вторично)
- Верстак: зелья, заточка  
- **Camp services (Фаза 4):** remove / upgrade карты за золото (как StS)  
- Не обязателен для прохождения

## 6. Цикл одного спуска (run)

```
ХАБ (лагерь Совета)
  → выбор 3 героев / сложности
  → GENERATED LABYRINTH (seed)
  → исследование коридоров
  → пачка мобов → КАРТОЧНЫЙ БОЙ
  → победа → DRAFT СЛОЙ 1 (карта ИЛИ skip→gold/minor)
  → (если level-up) → DRAFT СЛОЙ 2 (upgrade / rare)
  → сундуки / тупики / секреты / элиты
  → БОСС ЭТАЖА → DRAFT СЛОЙ 3 (relic + shard + optional curse altar)
  → следующий этаж или возврат в хаб
  → (хаб) remove/upgrade за золото
```

## 7. Система карт и прогрессия (зафиксировано)

Классика **roguelite deckbuilder** (Slay the Spire–like) в **first-person лабиринте**.

### 7.1 Типы карт
**Attack · Guard · Skill · Spell** (+ **Curse** с алтаря)  
**Relic** — не карта колоды, а **постоянная пассивка на забег**  
**Shard (Осколок)** — мета-прогрессия между забегами (не в колоду)

### 7.2 Колоды
| Правило | Значение |
|---------|----------|
| Колод на забег | **3** (по одной на героя) |
| Пул карт | Свой под класс/роль |
| Старт колоды | **8–10** карт на героя |
| Soft comfort | ~**15–18** |
| Hard pressure | >**20–22** — повод идти в remove (штраф опционально позже) |
| Remove / burn | **В хабе за золото** (Фаза 4); на этаже — редкие события |

### 7.3 Три слоя выбора (не смешивать экраны)

#### Слой 1 — После убийства пачки (обычный бой)
- На выбор **2–3** новые карты (атака / блок / способность) из пула **героя, который «вёл» бой** или из пула **всех троих** (бейдж класса)  
- Берёшь **одну** в соответствующую колоду  
- **Или skip** ради другой награды: **+золото** / малый шанс common-реликвии (~10%)  
- **Не** давать и карту, и сильную реликвию с каждой пачки (чтобы босс не обесценился)

```
[ Card A ] [ Card B ] [ Card C ]
        [ Skip → +gold ]   ·   [ Skip → minor relic roll ]
```

#### Слой 2 — После level-up персонажа
- **Отдельный** экран (другая рамка/цвет: золото / фиолетовый)  
- **Не** смешивается с post-combat draft  
- Выбор:  
  - **Upgrade** существующей карты (`Slice` → `Slice+`, больше урона / дешевле)  
  - **ИЛИ** мощная **новая** карта выше обычного дропа (uncommon/rare class card)  
- Ось «сила героя», не «лут с моба»

#### Слой 3 — После босса этажа
- **Реликвия:** pick **1 of 2–3** (постоянный баф на забег)  
- **Осколок Печати / Короны:** **auto** → мета между забегами  
- **Алтарь проклятия (optional):** сильная карта **с** побочкой (Curse / downside) **или skip**  
- Не сваливать всё в инвентарь без выбора — игрок **думает**

### 7.4 UI / энергия (решение на прототип)
- **Одна рука** в first-person UI (как реф)  
- Карты помечены **героем** (цвет / иконка)  
- Энергия: сначала **общие 3⚡ на ход отряда**; если слабо — **1⚡ на героя**  

### 7.5 Примеры карт MVP

| Карта | Герой | ⚡ | Эффект |
|-------|-------|---|--------|
| Slash | любой melee | 1 | 6 урона |
| Hack | танк/берсерк | 2 | 10 урона |
| Shield | танк | 1 | +5 Block |
| Firebolt | маг | 1 | 5 урона, ignore 1 armor |
| Rally | support | 0 | +1 ⚡ next turn (1/ход) |
| Slice+ | upgrade | 1 | 9 урона (пример апгрейда) |

### 7.6 Порядок внедрения (чтобы не разорваться)
1. 1 герой + 1 колода + post-combat draft (слой 1)  
2. Level-up draft (слой 2)  
3. 3 героя × 3 колоды  
4. Boss relics + shards + curse altar (слой 3)  
5. Camp remove/upgrade за gold

## 8. Генерация лабиринта (технически)

### Алгоритм MVP (Фаза 1)
1. Сетка `W × H` (например 41×41 клеток)  
2. Расставить `N` прямоугольных **комнат** без сильного overlap  
3. Соединить **MST / sequential L-corridors**  
4. Пометить **тупики** (клетки коридора с 1 соседом)  
5. **Двери** — на стыке room↔corridor  
6. **Сундук** — в 1 тупике или дальней комнате  
7. **Факелы** — вдоль стен с шагом  
8. **Пачки** — в N комнатах (не старт), маркер Encounter  
9. **Босс-комната** — самая дальняя от старта (Фаза 2 можно усилить)

### Содержимое клетки
`EMPTY | FLOOR | WALL | DOOR | CHEST | TORCH | ENCOUNTER | START | EXIT`

## 12. Решения, которые зафиксированы

1. Бой = **карты + энергия + end turn** (не Doom-shooter).  
2. Исследование = **1st person коридоры**.  
3. Крафт/еда = **не core**.  
4. Партия = **3 героя, колода у каждого** (свои пулы).  
5. **3 слоя прогрессии карт:** пачка → level-up → босс (экраны не смешивать).  
6. После пачки: draft **или** skip (gold / minor), не «карта + сильная реликвия».  
7. После босса: relic **pick**, shard **auto**, curse altar **optional**.  
8. Remove/upgrade за gold — **в хабе** (Фаза 4), soft cap колоды ~18.  
9. Арт = cartoon dark fantasy; AI-генерация ок + style lock.  
10. Название рабочее: **Ashen Depths**.
